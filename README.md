# Color Wheel for VS Code

A Visual Studio Code extension for creating harmonious color palettes without leaving the editor.

Choose a base color, generate combinations using established color harmony rules, and save your favorite palettes for use across projects.

## MVP Features

- Choose a base color with a color picker or enter a HEX value
- Generate complementary, analogous, triadic, and split-complementary palettes
- Preview colors and their HEX values
- Copy individual colors to the clipboard
- Save favorite palettes between VS Code sessions
- Work from a dedicated VS Code sidebar

## Planned Features

- Perceptually balanced palette generation with OKLCH
- WCAG contrast checking
- Insert colors directly into the active editor
- Store palettes globally or per workspace
- Export palettes as CSS custom properties, SCSS, JSON, and Tailwind CSS
- Generate color scales and design tokens
- Preview palettes on common UI components

## Proposed Technology

- TypeScript
- VS Code Extension API
- Webview API
- HSL for color harmony geometry
- OKLCH for perceptually balanced palettes

## Project Status

The project is currently in the MVP planning stage.
